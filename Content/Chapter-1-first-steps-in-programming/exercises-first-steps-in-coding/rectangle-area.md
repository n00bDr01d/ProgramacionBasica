# Problema: Calcular el área del rectángulo

Escriba un programa de C# que **lea** desde la consola **dos números**, a y b, **calcule** e **imprima** el área de un rectángulo con los lados **a** y **b**.

**Entrada y salida de muestra**

| un | b | área |
| -- | - | ---- |
| 2  | 7 | 14   |
| 7  | 8 | 56   |
| 12 | 5 | 60   |
